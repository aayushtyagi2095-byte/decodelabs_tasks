# Data Analytics Excel Project

## Overview

This project analyzes sales data using Microsoft Excel. It includes data cleaning, pivot tables, charts, an interactive dashboard, and business insights.

## Features

* Data Cleaning
* Pivot Tables
* Interactive Dashboard
* KPI Analysis
* Sales Trends
* Product Performance Analysis
* Business Insights

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard
* Conditional Formatting

## Project Outcome

The dashboard provides insights into sales performance, top-selling products, payment methods, and monthly trends to support business decision-making.
